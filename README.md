# 📊 Sales Dashboard Project

## 🧾 Overview

This project presents a **Sales Dashboard** created to analyze and visualize key business metrics for a company. The dashboard provides insights into sales performance, customer behavior, and revenue trends to support data-driven decision-making.

---

## 🎯 Objectives

* Track overall sales performance
* Identify top-performing products and regions
* Analyze revenue trends over time
* Support business decisions with clear visual insights

---

## 📁 Dataset

The dataset used in this project contains sales-related information such as:

* Order Date
* Product Category
* Region / Location
* Sales Revenue
* Profit / Cost

*(Dataset was sourced and prepared for analysis in CSV/Excel format.)*

---

## 🛠 Tools Used

* Microsoft Power BI
* Microsoft Excel (for data cleaning and preparation)

---

## 📈 Key Features of the Dashboard

* **Sales Overview**: Total revenue, profit, and key metrics
* **Time Analysis**: Monthly and yearly sales trends
* **Product Performance**: Best and worst-performing products
* **Regional Analysis**: Sales distribution by location
* **Interactive Filters**: Slicers for dynamic data exploration

---

## 📊 Insights

* Identified top-performing products contributing the highest revenue
* Observed seasonal trends in sales performance
* Detected underperforming regions needing improvement
* Highlighted opportunities for business growth

---

## 🚀 How to Use

1. Open the Power BI file (.pbix)
2. Explore the dashboard using filters and visuals
3. Interact with charts to gain deeper insights

---

## 📌 Conclusion

This Sales Dashboard helps transform raw data into meaningful insights, enabling better strategic planning and performance tracking for the business.
